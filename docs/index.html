FixCraft Purple Edition

Secure, cross-compatible encryption for files and media.

BASEFWX pairs post-quantum key wrapping with AES-GCM, ships a native C++ core, and keeps Python/C++ formats interoperable. Use it for file encryption, jMG media scrambling, and reversible data workflows.

Read README CLI Reference

Quick Start

pip install basefwx python -m basefwx cryptin aes-light file.bin -p "pass" python -m basefwx cryptin aes-light file.bin.fwx -p "pass"
